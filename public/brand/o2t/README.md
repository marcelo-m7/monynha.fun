# O2T Brand Pack

Identidade visual proposta para Open 2 Technology (O2T), com linguagem clean, futurista e orientada a produto digital.

## Conceito principal (selecionado)

Conceito 1: Flow Bridge (implementado no pacote principal)
- Ideia: o "O" representa abertura e ecossistema; o "2" nasce como fluxo de dados; o "T" funciona como destino e estrutura.
- Direcao visual: geometria limpa, cantos suaves e gradiente azul-violeta para sinalizar tecnologia global.
- Resultado: assinatura forte para web, app e materiais institucionais.

## Conceitos alternativos

Conceito 2: Circuit Wave
- Ideia: conexao em trilha de circuito com nodos, enfatizando interoperabilidade e engenharia.
- Arquivo: concepts/o2t-concept-2-circuit-wave.svg

Conceito 3: Open Portal
- Ideia: "portal aberto" com trilha interna para o 2, sugerindo plataforma e expansao.
- Arquivo: concepts/o2t-concept-3-open-portal.svg
